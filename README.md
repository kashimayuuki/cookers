# COOKERS

## サイト概要
### サイトテーマ
作った料理の情報交換をするサービス

### テーマを選んだ理由
自分が一人暮らしを始めた時に
外食は高い、冷凍食品ばかりとなり
自分で料理を始め、より美味しいものを
作りたいと思ったのがきっかけです。

手早く作れるもの、凝った物などの情報を
共有できるサービスがあれば解決できるのでは
と思い作成しました。

### ターゲットユーザ
*料理を作っている方
*料理を始める方
